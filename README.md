Surf is a fork of [LeeesExploitFixer](https://github.com/XeraPlugins/LeeesExploitFixer-3.0) of a fork of 254nm's [L2X9Core](https://github.com/254nm/L2X9Core)

- Support 1.7.x - 1.20.x (maybe?)
- Folia support
